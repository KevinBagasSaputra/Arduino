    Cara Pertama (Basic yang dipelajari)

    Kabel Jumper (2) Kabel Usb Type B (1) Sisahan Kabel Jumper yang Sudah ditancap
    Dimasukkan pada Arduino Uno R3, karena terdapat 2 unit yang akan dipasangkan.
    Maka unit 1, di pasang ke lubang 3.3V dan unit ke 2 Gnd.     		Tancapkan kabel USB Type B ke Arduino, kemudian kabel satunya lagi.
    yaitu USB Type A masukkan pada perangkat Laptop atau Komputer. Sesuai dengan perangkat yang kalian gunakan.     		Karena kabel jumper sebelumnya sudah tertancap pada masing-masing port.
    Sekarang terdapat 2 kabel Jumper yang belum tertancap, maka kita tancapkan pada Beardboard.
    3.3V => Positif, Gnd => Negatif

    # Led terdapat panjang kaki yang berbeda, masing-masing kaki memiliki.
    Makna tersendiri;

  • Panjang = Positif ( + )
  • Pendek = Negatif ( - )

    • Tahap Akhir

    Led (1)
    Tancapkan 1 buah LED pada Breadboard.
    Kaki panjang tancapkan ke kutub positif.
    Kaki pendek tancapkan ke kutub negatif.
    sertakan penempatan LED tersebut sesuai tempat yang semestinya.

    Cara Kedua (Menyalakan pada Arduino Uno R3)

    Kabel Usb Type B (1) Led
    Masukkan pada interface Type B Arduino Uno R3.
    Kemudian Kabel Type A ke interface Laptop.     		Tancapkan Led pada.
    DIGITAL (PWM ~).
    Pendek = 0 (Negatif)
    Panjang = 1 (Positif).

    Lanjutannya (Lakukan pada Software Arduino IDE)

    Void Setup()
    { #define lampu_1_positif 1 //1 ini berarti interface led yang kita colokkan yaitu 1(Positif / Panjang) #Artinya kita mendeklarasikan Variabel
    #define lampu_0_negatif 0 //0 juga interface led yang kita colokkan yaitu 0(Negatif / Pendek)
    pinMode(lampu_1_positif, OUTPUT);
    pinMode(lampu_0_negatif, OUTPUT); }

    Void loop()
    { digitalWrite(lampu_1_positif, HEIGH);
    digitalWrite(lampu_0_negatif, LOW;
    delay (100);
    digitalWrite(lampu_0_negatif, HEIGH);
    digitalWrite(lampu_1_positif, LOW);
    delay (100); }
